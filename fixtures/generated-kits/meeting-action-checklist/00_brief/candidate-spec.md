# Candidate Spec: Meeting Action Checklist Assistant

## Intended use

Help a user convert pasted meeting notes into a structured, reviewable checklist without integrating with live calendars, email, task tools, or team systems.

## Proposed classification

`skill`

The capability is narrow, reusable, and does not need isolated memory, credentials, scheduling, or a new profile distribution.

## User-owned material

Synthetic example meeting notes were supplied for the fixture. The user declined additional research, examples, SOPs, or private workspace material for v1.

## Operating model

`ICM-style`

The assistant should run as a staged, reviewable draft flow. It does not need real-time monitoring or external side effects.

## Review prompt

Please choose one: approve this skill-style direction, amend the checklist fields, add more examples, switch to an existing-agent patch, or stop. Approval of this spec does not approve installing files or changing any live profile.
